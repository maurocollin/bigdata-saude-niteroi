# Análise de Indicadores de Saúde: Niterói/RJ 🏥

Projeto de Big Data focado na análise da oferta hospitalar e identificação de vazios assistenciais.

## 🚀 Estrutura do Projeto
1. **Coleta**: Dados oficiais do DATASUS via PySUS e planilhas municipais.
2. **Limpeza**: Normalização com Pandas (Tratamento de Strings e CEPs).
3. **Veracidade**: Saneamento de dados irrelevantes e verificação territorial.
4. **Estatística**: Cálculo de Média, Mediana e Desvio Padrão para medir a desigualdade.
5. **Outliers**: Identificação de anomalias e polos de saúde via Z-Score.

## 🛠️ Tecnologias Utilizadas
* Python (Pandas, Scipy, Matplotlib)
* Folium (Mapas Interativos)
* Geobr (Malhas do IBGE)
